STEP BY STEP

This tutorial will guide you through the process of obfuscating your Javascript files.

After running Javascript Obfuscator you will see the main screen of the program.


You can either create a new project or use an already existing one. To open your project you should click "Open Project" button on the Toolbar or select "File/Open Project" in the Main menu.
To reset the existing project you should click "tbNew" button on the Toolbar or select "File/Reset All" in the Main menu.


1) First of all you should open files you want to obfuscate in Javascript Obfuscator. To add html page or some .js files you should click "Add File" button on the Toolbar or select "Project/Add File" in the Main menu. You should click the file you want to delete and then click "Delete File" button on the Toolbar or select "Project/Delete File" in the Main menu.

The window “Open dialog” will appear, in which you can select html or js files. Javascript Obfuscator automatically opens all .js files associated with an html page.

2) You can also set some options to reduce the file size and minimize the loading time. If you want to delete comments, white spaces or change names set the corresponding checkboxes on the " Options"panel. If the checkbox Change Names is set the Javascript Obfuscator will rename variables and functions. With Names in Strings option the Javascript Obfuscator will additionally try to find names in string, not in the code. It can be useful if your code generates some commands from strings, for example by "eval" method. Be careful with this option.

3) Read names from the added files. To do it click "Read Names" button on the "Options" panel or select "Project/Read Names" in the Main menu. In the Rename Panel you will see all your Variables, Objects, Functions. You can also see the number of Functions and Variables entries in a script.

4)If you want to change the name of some variable by obfuscating, mark it and fill "New Name" field for it. You can do it in a two ways:

à) Automatically. To rename it in Auto mode you should at first mark the names in the Rename Panel. To do it use the right-click context menu. You can also add or delete new names using context menu. Then you should click "Generate names" button on the "Options" panel or select "Project/ Generate Names " in the Main menu. You'll see that all marked names will have new names.

 

  á) Manually. If you want to change Variable or Function you should double-click the required name or right-click the required name and select "Edit Name" in the context menu. "Enter new Name" dialog will appear. You should type new Name of Variable or Function.

5) If you've created new names for all the required names and set all the options you can start process of obfuscating the data.

To do it you should click "Process Project files" button on the Toolbar or on the right panel or select "Process and save files" in the Main menu.

You can also view the statistic.

6) After the obfuscating all initial files will be replaced with obfuscated files.

7) Now it is time to save your project file. Notice that you are free to save your project at any time. To save the project just click "Save Project" button on the Toolbar or select "Save Project" in the Main menu. The "Save as…" dialog will appear, in which you should select a place you want the project to be saved in, and enter the project name. After clicking "Ok", the project will be saved.
